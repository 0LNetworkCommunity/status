# Status
A tool to check the status of a OL network. This is used as a test tool to easily and quickly verify changes within the network. 


## To Deploy
```
yarn
yarn build
yarn deploy

```

## To Dev
```
yarn
yarn dev

```

## API Views

### System
- transaction fees

#### block info
- epoch length
- block height

### Validators
